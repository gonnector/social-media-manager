# Social Media Manager

AI-powered social media posting automation tool. Automates cross-platform content translation, formatting, and posting via browser automation.

## Target Platforms

- **Threads** (Meta)
- **X** (Twitter)
- **LinkedIn**

## Features (Planned)

- [ ] Cross-platform text posting automation
- [ ] Image attachment automation
- [ ] Multi-language content translation
- [ ] Platform-specific formatting (character limits, thread chains)
- [ ] LinkedIn Article integration
- [ ] Threads thread chain composition

## Architecture

Browser automation (claude-in-chrome MCP) + PowerShell clipboard bridge for text/image transfer.
