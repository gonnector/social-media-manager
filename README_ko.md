# Social Media Manager

AI 기반 소셜 미디어 포스팅 자동화 도구. 브라우저 자동화를 통해 크로스 플랫폼 콘텐츠 번역, 포맷팅, 게시를 자동화합니다.

## 대상 플랫폼

- **Threads** (Meta)
- **X** (Twitter)
- **LinkedIn**

## 기능 (계획)

- [ ] 크로스 플랫폼 텍스트 포스팅 자동화
- [ ] 이미지 첨부 자동화
- [ ] 다국어 콘텐츠 번역
- [ ] 플랫폼별 포맷팅 (글자 수 제한, 스레드 체인)
- [ ] LinkedIn Article 연동
- [ ] Threads 스레드 체인 구성

## 아키텍처

브라우저 자동화 (claude-in-chrome MCP) + PowerShell 클립보드 브릿지로 텍스트/이미지 전달.
